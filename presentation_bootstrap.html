Bootstrap

Qu'est-ce que Bootstrap?

  • Bootstrap est un front-end framework libre pour le développement web plus rapide et plus facile
  • Bootstrap comprend HTML et des modèles de conception à base de CSS pour la typographie, des formes, des boutons, tables, navigation, modaux, carrousels d'image et bien d'autres, ainsi que des plugins en option JavaScript
  • Bootstrap vous donne également la possibilité de créer facilement des conceptions sensibles

Historique

  • Avant l'arrivée de Bootstrap, plusieurs bibliothèques existaient, ce qui menait à des incohérences et à un coût de maintenance élevé. Cette plate-forme a été conçue par deux développeurs faisant partie de la mouvance de développeurs qui gravitent autour de Twitter, Mark Otto et Jacob Thornton, et avait le nom de Twitter Blueprint en 2010. Le premier déploiement à échelle réelle eut lieu lors de la première hackweek organisée par Twitter. En août 2011, Twitter place Bootstrap sous licence open source. En février 2012, Bootstrap est le projet le plus populaire sur GitHub.
>

Pourquoi utiliser Bootstrap?

  • Facile à utiliser: Tout le monde avec seulement des connaissances de base du langage HTML et CSS peut commencer à utiliser Bootstrap
  • Fonctionnalités réactives: CSS réactives de Bootstrap se adapte aux téléphones, les tablettes et les ordinateurs de bureau
  • Approche mobile d' abord: Dans Bootstrap, styles mobile premiers font partie du cadre de base
  • Compatibilité du navigateur: Bootstrap 4 est compatible avec tous les navigateurs modernes (Chrome, Firefox, Internet Explorer 10+, Edge, Safari et Opera)

  • CDN est l'acronyme de « Content delivery network » ; c'est un réseau de serveurs qui met à disposition des librairies. Il devient ainsi inutile de prévoir ces librairies sur son propre serveur, il suffit de « pointer » vers eux. Il y a des avantages à utiliser un CDN :

    • Actualisation automatique des librairies ;
    • Amélioration du référencement…
